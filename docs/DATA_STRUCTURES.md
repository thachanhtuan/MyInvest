# Структуры данных

---

## Перечисления (Enum)

### AccountType

Тип счёта.

| Значение  | Описание                                 |
| --------- | ---------------------------------------- |
| `broker`  | Брокерский счёт                          |
| `iis`     | Индивидуальный инвестиционный счёт (ИИС) |
| `deposit` | Банковский депозит                       |
| `savings` | Накопительный счёт                       |

### AssetClass

Класс актива (используется в целевой аллокации).

| Значение        | Описание                           |
| --------------- | ---------------------------------- |
| `stock_ru`      | Российские акции                   |
| `stock_foreign` | Иностранные акции                  |
| `bond_gov`      | Государственные облигации (ОФЗ)    |
| `bond_corp`     | Корпоративные облигации            |
| `bond_muni`     | Муниципальные облигации            |
| `etf`           | Фонды (ETF / БПИФ)                 |
| `deposit`       | Банковские депозиты                |
| `cash`          | Денежные средства / денежный рынок |
| `gold`          | Золото                             |
| `other`         | Прочее                             |

### TransactionType

Тип транзакции.

| Значение          | Описание                                  |
| ----------------- | ----------------------------------------- |
| `buy`             | Покупка                                   |
| `sell`            | Продажа                                   |
| `dividend`        | Дивиденд                                  |
| `coupon`          | Купон по облигации                        |
| `amortization`    | Частичное погашение облигации             |
| `maturity`        | Полное погашение облигации                |
| `commission`      | Комиссия брокера                          |
| `tax`             | Налог                                     |
| `deposit_in`      | Пополнение счёта                          |
| `deposit_out`     | Вывод со счёта                            |
| `interest`        | Проценты по депозиту                      |
| `initial_balance` | Первоначальный баланс (начальная позиция) |
| `other`           | Прочее (подарки, промо-акции)             |

### Currency

| Значение | Описание         |
| -------- | ---------------- |
| `RUB`    | Российский рубль |
| `USD`    | Доллар США       |
| `EUR`    | Евро             |
| `CNY`    | Юань             |
